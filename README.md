# 🛒 E-Commerce Data Analysis using Python & SQL
📌 Project Overview
This project analyzes an e-commerce dataset using **Python and SQL** to extract meaningful business insights.
The goal is to understand customer behavior, sales trends, and product performance.
🎯 Problem Statement

Analyze e-commerce transaction data to:

* Identify sales trends over time
* Find top-performing products
* Understand customer purchasing behavior
* Generate insights to improve business decisions
🛠️ Tech Stack
* 🐍 Python (Pandas, Matplotlib, Seaborn)
* 🗄️ SQL (Joins, Aggregations, Group By)
* 📓 Jupyter Notebook
  
📂 Dataset Information
The dataset contains:

* Invoice details
* Product descriptions
* Quantity and pricing
* Customer information
* Country data

🔍 Key Analysis Performed

 📊 1. Data Cleaning
* Removed null values
* Handled incorrect data types
* Filtered invalid records

📈 2. Sales Analysis
* Total revenue calculation
* Monthly and yearly sales trends
* Peak sales periods

🛍️ 3. Product Analysis
* Top-selling products
* Most profitable categories

👥 4. Customer Analysis
* Top customers by purchase
* Country-wise customer distribution

📊 Sample Insights

* 📌 Sales increased significantly during festive seasons
* 📌 Certain products contribute to the majority of revenue
* 📌 A small group of customers drives high sales

🚀 How to Run the Project

1. Clone the repository
2. Open the Jupyter Notebook
3. Install required libraries:

   ```bash
   pip install pandas matplotlib seaborn
   ```
4. Run all cells
